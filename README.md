# SlashBlade_Resharped_Fork
这是一个拔刀剑：重锋的fork版本，原项目地址：https://github.com/0999312/SlashBlade_Resharped

制作这个版本的起因是向玩工业时代2经典版（IC2C）和拔刀剑。但是最初版本的拔刀剑机制过于复杂且没有明确的路线说明，以及没有兼容jei的配方。所以选择了拔刀剑：重锋。但是因为原作者明确说明了不会兼容1.19.2版本，所以我用ai创建了这个版本

因为本人没有模组开发经验，项目修改全部通过AI完成（~~编程工具：Trae，官网：https://www.trae.cn~~改用codex，GPT牛B）项目中一定有很多的垃圾代码和无用注释，因此不建议fork此版本进行修改或提交issue

目前项目仍未完全兼容1.19.2版本，仍然存在很多的bug。还在拷打AI中，应该、大概、可能可以在生存模式尝试游玩了。
